# WildINTEL Misc Documentation

![WildINTEL](img/wildIntel_logo.webp){ style="display: block; margin: 0 auto;" }

Welcome to the WildINTEL Misc documentation.

**WildINTEL Misc** is a general-purpose repository within the
[WildINTEL project](https://wildintel.eu/) used to store resources, assets, scripts,
and other artefacts that do not belong to a specific component repository. **It has been
developed within the framework of the WildINTEL project, funded by Biodiversa+ under the
2022-2023 BiodivMon joint call. It has been co-funded by the European Commission (GA No.
101052342) and the following funding organisations: Agencia Estatal de Investigación
(Spain, PCI2023-145963-2, PCI2024-153489), National Science Centre (Poland,
UMO-2023/05/Y/NZ8/00104), the Research Council of Norway (Norway, NFR350962) and the
German Research Foundation (Germany).**

This repository acts as a shared space for the WildINTEL team to keep configuration
snippets, documentation drafts, utility scripts, reference data, and any other material
that supports the project without fitting neatly elsewhere.
